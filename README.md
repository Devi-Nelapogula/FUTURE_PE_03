# SEO Content Cluster Project

This project is a simple and responsive website created using HTML, CSS, and JavaScript.

It is based on an **SEO Content Cluster strategy** for a local business website.  
The project shows how AI, SEO, and Prompt Engineering can be used together to build structured blog content for better Google ranking, traffic, and leads.

---

## Project Objective

The main goal of this project is to create a website page that explains an SEO content strategy for a business.

This strategy includes:

- one strong pillar blog
- multiple supporting blogs
- local SEO keywords
- internal linking strategy
- keyword-based content planning

---

## Business Chosen

**SmileCare Dental Clinic**  
**Location:** Bangalore  
**Business Type:** Dental Clinic

This project is designed as if the SEO content will be published on a real dental clinic website.
